Installing and Running
----

Clone GitHub repo:

```
git clone https://github.com/kmikodev/dynamic-components.git
```

Install node module dependencies:

```
npm i 
```

Run application:

```
npm start
```

Go to [http://localhost:4200](http://localhost:4200) in your browser.
